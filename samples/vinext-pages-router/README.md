# Vinext Pages Router Sample

This is a sample of using NextJS pages router on Spin using Vinext. This is a sample ported from the [example](https://github.com/cloudflare/vinext/tree/main/examples/pages-router-cloudflare) in the [Vinext](https://github.com/cloudflare/vinext).

## Building and Running

Build the app using:

```bash
spin build
```

Run the app using:

```bash
spin up
```

The app can then be accessed at `http://localhost:3000`
